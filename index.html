Configuration

Csstyle-button has some variables which are possible to edit, it's really simple :

Variable Description Default Example
$csstyle-button-namespace The namespace of the component button ".btn"
$csstyle-button-font-family The font family to use "Helvetica Neue", Helvetica, Arial, sans-serif "Lato"
$csstyle-button-font-weight The font weight to use 300 400
$csstyle-button-base-font-size The base font size to use 15px 17px
$csstyle-button-base-padding-top-bottom The top and bottom padding to use 6px 8px
$csstyle-button-base-padding-right-left The right and left padding to use 12px 14px
